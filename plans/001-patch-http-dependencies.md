## Why this matters

The repository locks versions of Bandit, Plug, Phoenix, and hpax with published, remotely reachable denial-of-service vulnerabilities. This application exposes Bandit directly, enables both LiveView WebSocket and long-poll transports, and installs body parsers before routing, so the vulnerable code paths are part of the public surface. Patch the compatible dependency lines immediately without combining this work with a Phoenix 1.8 or LiveView major migration.

Relevant advisories include:

- Bandit chunked-body denial of service: <https://github.com/mtrudel/bandit/security/advisories/GHSA-9q9q-324x-93r2>
- Bandit fragmented WebSocket memory exhaustion: <https://github.com/mtrudel/bandit/security/advisories/GHSA-pf94-94m9-536p>
- Plug multipart parser denial of service: <https://github.com/elixir-plug/plug/security/advisories/GHSA-468c-vq7p-gh64>
- Phoenix long-poll memory exhaustion: <https://github.com/phoenixframework/phoenix/security/advisories/GHSA-628h-q48j-jr6q>
- Phoenix unbounded channel joins: <https://github.com/phoenixframework/phoenix/security/advisories/GHSA-6983-jfq8-485w>
- hpax unbounded HPACK integer decoding: <https://github.com/advisories/GHSA-jj2p-32j7-whj2>

Use this version gate after resolving and compiling:

```bash
mix run --no-start -e '
minimums = %{bandit: "1.11.1", hpax: "1.0.4", phoenix: "1.7.24", plug: "1.19.5"}

Enum.each(minimums, fn {app, minimum} ->
  _ = Application.load(app)
  actual = app |> Application.spec(:vsn) |> to_string()

  if Version.compare(actual, minimum) == :lt do
    raise "#{app} #{actual} is below required #{minimum}"
  end

  IO.puts("#{app}=#{actual}")
end)
'
```

Expected: exit 0 and one version line for each of `bandit`, `hpax`, `phoenix`, and `plug`, all at or above the minimums above.
